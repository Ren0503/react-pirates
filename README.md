## Install

```bash
npm install --save react-pirates
```

## Features

react-pirates help you perform a lot of work about advanced react hooks

### useAsync

For handle asynchronous calls

#### Usage

```jsx
  const imgFetch = useAsync(url =>
    fetch(url).then(response => response.json())
  );
```
